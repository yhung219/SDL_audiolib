Detailed Description

Sample-based audio playback stream.

This class handles playback for sample-based, digital audio streams produced by an AudioDecoder.

◆ seekToTime()

bool Aulib::AudioStream::seekToTime ( float  seconds)
overridevirtual

Seek to a position in the stream.

This will change the current playback position in the stream to the specified time, in seconds.

Note that for some streams (for example MOD files and some MP3 files), it might not be possible to do an accurate seek, in which case the position that is set might be off by some margin. In some streams, seeking is not possible at all (MIDI files, for example.)

Parameters
secondsPosition to seek to.
Returns
Return values
trueThe playback position was changed successfully.
falseThis stream does not support seeking.

Change playback volume.

Parameters
